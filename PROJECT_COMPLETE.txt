🚀 编译与部署
━━━━━━━━

📍 项目位置：
   /sdcard/project/HardcoreRevive/

🛠️ 编译命令：
   cd /sdcard/project/HardcoreRevive
   mvn clean package

📦 输出文件：
   target/HardcoreRevive-1.0.0.jar

💾 安装步骤：
   1. 复制JAR到服务器 plugins/ 目录
   2. 启动/重启服务器
   3. 编辑 plugins/HardcoreRevive/config.yml
   4. 使用 /hcreload 重载配置

🎮 基本命令速查
━━━━━━━━

  玩家命令：
    /revive <码>              - 使用复活码
    /revive list              - 查看复活码
    /sos                      - 发送求救
    /trust add <玩家>         - 添加信任
    /bounty create <描述>     - 创建悬赏

  管理命令：
    /setjail                  - 设置小黑屋
    /reviveadmin generate     - 生成复活码
    /reviveadmin gui          - 管理界面
    /reviveadmin forceRevive  - 强制复活
    /hcreload                 - 重载配置
